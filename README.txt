Customize ACF Flexible field with zero coding.

== Description ==

Customize ACF Flexible field with zero coding. Like we said, just choose layouts you want, and you're good to go.

== Installation ==

1. Upload `acf_ffc` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
